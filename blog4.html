Media queries are CSS rules that only get executed under certain conditions. They are counted as being part of the CSS code, but they are "removed" from the CSS rule list when a certain condition is not met.

Start it with @media screen (to affect the normal site layout, not the print layout) and (conditions here. Typically max or min-width) { put normal CSS inside of here } Example:


@media screen and (min-width: ) {
display: flex;
}
.item {
height: 100px;
width: 100px; /* A fixed width as the default */
}
.item-center {
flex-grow: 1; /* Set the middle element to grow and stretch */
}
.item + .item {
margin-left: 2%;
}

This is kind of one of those "best practice" things, but the way I structure media queries is with min-width instead of max-width. That way my base styles are mobile and the media queries handle progressively larger sizes. Helps to match that "mobile-first design" paradigm.
