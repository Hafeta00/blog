Generally speaking, CSS is a website building tool, which allows you to make changes to your website easier and faster. You don't need to edit every sub-page of the website, you only need to edit 1 file that is referenced on all sub-pages. CSS also allows the use of more advanced techniques and the creation of unique website designs. CSS or Cascading Style Sheets is a coding language that controls the layout of your website. This can include things such as background color or image, table layouts, font, etc. In the video refrenced it emphasizes that CSS heavily defines the appearance and style of a web page

CSS is basically a list of characteristics that you want to apply to every item of a certain type. If you want all paragraphs to be 16-point Helvetica, and to have half-inch margins at top and quarter-inch margins at the sides, you might write something like this:

p {
font-size: 16pt;
font-family: helvetica;
background-color: blue;
color: black;
margin-top: .5in;
margin-left: .25in;
margin-right: .25in;
}

You write that CSS information in another text file and you tell your html file to always get style information from the CSS file. You can have multiple different pages that point to one CSS file, so they all get their information from that CSS file. Every HTML file that uses that CSS file will have paragraphs with black text, a blue background, half-inch margins at the top, and quarter-inch margins at the sides.Because multiple HTML files can all use the same CSS file, you save a whole bunch of space. You don't have to specify that information over and over again on each page.
